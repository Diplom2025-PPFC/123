Лекція 4: Текстові поля TextBox та RichTextBox

Введення тексту, багаторядковий текст, форматування

⏱️ Тривалість: 3 години 📖 Лекція 4 з 55

4.1 TextBox

// Однорядкове текстове поле
TextBox^ txtName = gcnew TextBox();
txtName->Location = Point(120, 50);
txtName->Size = Size(200, 25);
txtName->Text = "Введіть ім'я";

// Багаторядкове текстове поле
TextBox^ txtMulti = gcnew TextBox();
txtMulti->Multiline = true;
txtMulti->ScrollBars = ScrollBars::Vertical;
txtMulti->Size = Size(300, 150);

// RichTextBox з форматуванням
RichTextBox^ rtb = gcnew RichTextBox();
rtb->Size = Size(400, 200);
rtb->AppendText("Звичайний текст\n");
rtb->SelectionColor = Color::Red;
rtb->SelectionFont = gcnew Font("Arial", 12, FontStyle::Bold);
rtb->AppendText("Червоний жирний текст\n");
