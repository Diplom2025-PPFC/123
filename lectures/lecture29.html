Лекція 29: Вступ до ООП: класи та об'єкти

Класи, об'єкти, члени класу, модифікатори доступу

⏱️ Тривалість: 3 години 📖 Лекція 29 з 60

29.1 Основні концепції ООП

Об'єктно-орієнтоване програмування (ООП) — це парадигма, що організовує код у вигляді об'єктів, які містять дані та методи для роботи з цими даними.

Чотири основні принципи ООП:

  • Інкапсуляція — приховування внутрішньої реалізації
  • Успадкування — створення нових класів на основі існуючих
  • Поліморфізм — можливість об'єктів мати багато форм
  • Абстракція — виділення суттєвих характеристик

29.2 Визначення класу

#include <iostream>
#include <string>

class Rectangle {
// Приватні члени (доступні тільки всередині класу)
private:
    double width;
    double height;

// Публічні члени (доступні ззовні)
public:
    // Сеттери
    void setWidth(double w) {
        if (w > 0) width = w;
    }

    void setHeight(double h) {
        if (h > 0) height = h;
    }

    // Геттери
    double getWidth() const { return width; }
    double getHeight() const { return height; }

    // Методи
    double getArea() const {
        return width * height;
    }

    double getPerimeter() const {
        return 2 * (width + height);
    }
};

int main() {
    // Створення об'єкта
    Rectangle rect;
    rect.setWidth(5.0);
    rect.setHeight(3.0);

    std::cout << "Площа: " << rect.getArea() << std::endl;
    std::cout << "Периметр: " << rect.getPerimeter() << std::endl;

    return 0;
}

29.3 Підсумки лекції

  • Клас — шаблон для створення об'єктів
  • Об'єкт — екземпляр класу
  • private, public, protected — модифікатори доступу
  • Геттери/сеттери для контролю доступу
