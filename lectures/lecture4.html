Рис. 4.1 — Типи даних у C++

4.1 Що таке змінна?

Змінна — це іменована область пам'яті, що зберігає дані певного типу. Кожна змінна в C++ має:

  • Ім'я — унікальний ідентифікатор
  • Тип — визначає розмір та формат даних
  • Значення — фактичні дані
  • Адреса — місце в пам'яті
// Синтаксис оголошення:
// тип ім'я = значення;

int age = 25;           // ціле число
double pi = 3.14159;    // число з плаваючою крапкою
char letter = 'A';      // один символ
bool flag = true;       // булеве значення
std::string text = "Hello"; // рядок символів

4.2 Правила іменування змінних

  • Можуть містити літери (a-z, A-Z), цифри (0-9) та підкреслення (_)
  • Мають починатися з літери або підкреслення
  • Чутливі до регістру (Age та age — різні змінні)
  • Не можуть бути ключовими словами C++
  • Рекомендується використовувати змістовні імена
int studentAge;      // ✓ Правильно
int _privateVar;     // ✓ Правильно (допустимо, але уникайте)
int age2;            // ✓ Правильно
int 2ndValue;        // ✗ Помилка: починається з цифри
int my age;          // ✗ Помилка: пробіл не дозволено
int class;           // ✗ Помилка: ключове слово
int $money;          // ✗ Помилка: $ не дозволено

4.5 Символьний тип char

char letter = 'A';      // ASCII: 65
char digit = '7';       // ASCII: 55
char symbol = '$';      // ASCII: 36
char newline = '\n';    // символ нового рядка
char tab = '\t';        // символ табуляції

// Спеціальні символи (escape sequences)
// \n — новий рядок
// \t — горизонтальна табуляція
// \\ — зворотна коса риска
// \' — одинарні лапки
// \" — подвійні лапки
// \0 — нульовий символ
// \r — повернення каретки

4.6 Булевий тип bool

bool isReady = true;
bool isEmpty = false;

// bool конвертується в int: true = 1, false = 0
int flag = true;  // flag = 1

// Будь-яке ненульове значення — true, 0 — false
bool b1 = 100;    // true
bool b2 = 0;      // false
bool b3 = -5;     // true

4.7 Константи

Константи — це змінні, значення яких не можна змінити після ініціалізації.

// const — константа, визначена під час компіляції
const int MAX_SIZE = 100;
const double PI = 3.14159;
const char NEWLINE = '\n';

// constexpr — константний вираз (C++11)
constexpr int SQUARE(int x) { return x * x; }
constexpr int AREA = SQUARE(5); // AREA = 25

// #define — макрос препроцесора (устарілий спосіб)
#define MAX_VALUE 100
#define PI_VALUE 3.14159

// enum — перерахування
enum Color { RED, GREEN, BLUE };
const Color DEFAULT_COLOR = BLUE;

// enum class (C++11, типобезпечний)
enum class Status { OK, ERROR, WARNING };
Status s = Status::OK;

4.8 Літерали (Constants Values)

// Цілі літерали
int dec = 42;       // десяткове
int oct = 052;      // вісімкове (починається з 0)
int hex = 0x2A;     // шістнадцяткове (починається з 0x)
int bin = 0b101010; // двійкове (C++14, починається з 0b)

// Символьні літерали
char c1 = 'A';
wchar_t c2 = L'A';    // широкий символ
char16_t c3 = u'A';   // UTF-16 (C++11)
char32_t c4 = U'A';   // UTF-32 (C++11)

// Рядкові літерали
const char* s1 = "Hello";
const wchar_t* s2 = L"Hello";
const char* s3 = u8"Hello";  // UTF-8 (C++11)
const char* s4 = R"(Raw "string" with \ backslash)"; // Сирий рядок

4.9 Визначення власних типів

// typedef — псевдонім типу
typedef unsigned int uint;
typedef long double ldouble;
uint age = 25;  // те ж саме, що unsigned int

// using — сучасний спосіб (C++11)
using uint = unsigned int;
using ldouble = long double;
using StringList = std::vector<std::string>;

// auto — автоматичне визначення типу (C++11)
auto x = 10;        // int
auto y = 3.14;      // double
auto z = "Hello";   // const char*
auto v = std::vector<int>{1, 2, 3};

4.10 Підсумки лекції

  • Змінна — іменована область пам'яті з певним типом
  • Цілі типи: short, int, long, long long (зі знаком та без)
  • Дробові типи: float, double, long double
  • Символьний тип: char та його варіанти
  • Булевий тип: bool (true/false)
  • Константи: const, constexpr, enum, enum class
  • auto для автоматичного визначення типу
  • using/typedef для псевдонімів типів
