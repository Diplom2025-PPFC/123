Лекція 48: Файли: читання та запис

fstream, ofstream, ifstream, текстові та бінарні файли

⏱️ Тривалість: 3 години 📖 Лекція 48 з 60

48.1 Робота з файлами

C++ надає класи для роботи з файлами через бібліотеку <fstream>.

#include <iostream>
#include <fstream>
#include <string>

int main() {
    // Запис у файл
    std::ofstream outFile("data.txt");
    if (outFile.is_open()) {
        outFile << "Привіт, файл!" << std::endl;
        outFile << "Рядок 2" << std::endl;
        outFile.close();
    }

    // Читання з файлу
    std::ifstream inFile("data.txt");
    if (inFile.is_open()) {
        std::string line;
        while (std::getline(inFile, line)) {
            std::cout << line << std::endl;
        }
        inFile.close();
    }

    // Додавання в файл
    std::ofstream appendFile("data.txt", std::ios::app);
    appendFile << "Додано в кінець." << std::endl;
    appendFile.close();

    return 0;
}

48.2 Підсумки лекції

  • ofstream — запис, ifstream — читання
  • fstream — читання та запис
  • ios::app — додавання в кінець
