Лекція 16: Параметри функцій та return

Значення за замовчуванням, перевантаження, inline функції

⏱️ Тривалість: 3 години 📖 Лекція 16 з 60

16.1 Параметри за замовчуванням

Параметри за замовчуванням дозволяють викликати функцію без вказівки всіх аргументів. Параметри зі значеннями за замовчуванням мають бути останніми у списку.

#include <iostream>
#include <string>

// Параметри за замовчуванням
void greet(std::string name = "Світе", int times = 1) {
    for (int i = 0; i < times; i++) {
        std::cout << "Привіт, " << name << "!" << std::endl;
    }
}

// Математична функція
double power(double base, int exponent = 2) {
    double result = 1;
    for (int i = 0; i < exponent; i++) {
        result *= base;
    }
    return result;
}

int main() {
    greet();                  // Привіт, Світе!
    greet("Ivan");            // Привіт, Ivan!
    greet("Ivan", 3);         // Привіт, Ivan! (3 рази)

    std::cout << power(3) << std::endl;       // 9 (3^2)
    std::cout << power(2, 5) << std::endl;    // 32 (2^5)

    return 0;
}

16.2 Перевантаження функцій (Function Overloading)

Перевантаження дозволяє мати кілька функцій з однаковим іменем, але різними параметрами. Компілятор вибирає потрібну версію на основе аргументів.

#include <iostream>

// Перевантаження функції print
void print(int x) {
    std::cout << "Ціле число: " << x << std::endl;
}

void print(double x) {
    std::cout << "Дробове число: " << x << std::endl;
}

void print(const char* s) {
    std::cout << "Рядок: " << s << std::endl;
}

void print(int x, int y) {
    std::cout << "Два числа: " << x << ", " << y << std::endl;
}

int main() {
    print(42);           // Ціле число
    print(3.14);         // Дробове число
    print("Hello");      // Рядок
    print(10, 20);       // Два числа
    return 0;
}

16.3 Inline функції

Inline функція пропонує компілятору вставити тіло функції безпосередньо у місце виклику, замість звичайного виклику. Це зменшує накладні витрати на виклик функції.

// Inline функція
inline int square(int x) {
    return x * x;
}

// Еквівалентно:
// int result = square(5);
// Компілятор може замінити на:
// int result = 5 * 5;

#include <iostream>
int main() {
    std::cout << square(5) << std::endl;   // 25
    std::cout << square(10) << std::endl;  // 100
    return 0;
}

16.4 Підсумки лекції

  • Параметри за замовчуванням спрощують виклики
  • Перевантаження — одне ім'я, різні параметри
  • Inline функції — оптимізація для малих функцій
  • Return повертає результат функції
