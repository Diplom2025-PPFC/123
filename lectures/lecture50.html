Лекція 50: Простори імен (namespaces)

namespace, using, безіменні простори, вкладені простори

⏱️ Тривалість: 3 години 📖 Лекція 50 з 60

50.1 Простори імен

Простір імен — це область, що групує ідентифікатори для запобігання конфліктам імен.

#include <iostream>

// Власний простір імен
namespace Math {
    const double PI = 3.14159;

    double square(double x) {
        return x * x;
    }

    double circleArea(double r) {
        return PI * square(r);
    }

    namespace Advanced {
        double power(double base, int exp) {
            double result = 1;
            for (int i = 0; i < exp; i++) {
                result *= base;
            }
            return result;
        }
    }
}

namespace Utils {
    void printHello() {
        std::cout << "Привіт з Utils!" << std::endl;
    }
}

int main() {
    // Повні імена
    std::cout << "PI = " << Math::PI << std::endl;
    std::cout << "Площа кола: " << Math::circleArea(5) << std::endl;
    std::cout << "2^10 = " << Math::Advanced::power(2, 10) << std::endl;

    // using declaration
    using Math::square;
    std::cout << "4^2 = " << square(4) << std::endl;

    // using directive
    using namespace Utils;
    printHello();

    // Безіменний простір (видимий тільки у файлі)
    // namespace { ... }

    return 0;
}

50.2 Підсумки лекції

  • namespace — групування ідентифікаторів
  • Вкладені простори імен
  • using — для зручного доступу
