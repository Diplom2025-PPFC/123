Лекція 53: Розумні вказівники (smart pointers)

unique_ptr, shared_ptr, weak_ptr, автоматичне керування пам'яттю

⏱️ Тривалість: 3 години 📖 Лекція 53 з 60

53.1 Типи розумних вказівників

Розумні вказівники автоматично керують пам'яттю, запобігаючи витокам.

#include <iostream>
#include <memory>

class Resource {
public:
    Resource() { std::cout << "Resource створено" << std::endl; }
    ~Resource() { std::cout << "Resource знищено" << std::endl; }
    void use() { std::cout << "Використання ресурсу" << std::endl; }
};

int main() {
    // unique_ptr — унікальне володіння
    {
        std::unique_ptr<Resource> res = std::make_unique<Resource>();
        res->use();
    } // Автоматичне звільнення

    // shared_ptr — спільне володіння
    {
        std::shared_ptr<Resource> res1 = std::make_shared<Resource>();
        {
            std::shared_ptr<Resource> res2 = res1; // Копіювання
            std::cout << "Лічильник: " << res1.use_count() << std::endl; // 2
        } // Лічильник зменшується
        std::cout << "Лічильник: " << res1.use_count() << std::endl; // 1
    }

    return 0;
}

53.2 Підсумки лекції

  • unique_ptr — унікальне володіння
  • shared_ptr — спільне володіння з лічильником
  • weak_ptr — слабке посилання
