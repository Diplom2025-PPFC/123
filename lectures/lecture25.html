Лекція 25: Динамічна пам'ять: new та delete

Виділення та звільнення пам'яті, масиви в купі, витоки пам'яті

⏱️ Тривалість: 3 години 📖 Лекція 25 з 60

25.1 Оператори new та delete

Оператор new виділяє пам'ять у купі (heap) і повертає вказівник на неї. delete звільняє цю пам'ять.

#include <iostream>

int main() {
    // Виділення одного елемента
    int* ptr = new int;       // Виділення
    *ptr = 42;
    std::cout << *ptr << std::endl;
    delete ptr;               // Звільнення

    // Виділення з ініціалізацією
    int* p2 = new int(100);
    std::cout << *p2 << std::endl;
    delete p2;

    // Виділення масиву
    int size = 5;
    int* arr = new int[size];  // Динамічний масив
    for (int i = 0; i < size; i++) {
        arr[i] = i * 10;
    }
    delete[] arr;              // Звільнення масиву (delete[]!)

    return 0;
}

25.2 Підсумки лекції

  • new — виділення пам'яті в купі
  • delete — звільнення пам'яті
  • new[] / delete[] — для масивів
  • Завжди звільняйте виділену пам'ять!
