Лекція 23: Арифметика вказівників

Інкремент, декремент, віднімання, порівняння вказівників

⏱️ Тривалість: 3 години 📖 Лекція 23 з 60

23.1 Арифметичні операції з вказівниками

Арифметика вказівників дозволяє переміщуватися по масиву за допомогою інкременту/декременту вказівника.

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int* ptr = arr;  // Вказівник на перший елемент

    // Інкремент вказівника
    std::cout << *ptr << std::endl;    // 10
    ptr++;
    std::cout << *ptr << std::endl;    // 20
    ptr++;
    std::cout << *ptr << std::endl;    // 30

    // Додавання до вказівника
    int* p2 = arr + 3;  // Вказівник на arr[3]
    std::cout << *p2 << std::endl;    // 40

    // Віднімання вказівників
    int diff = p2 - arr;  // Кількість елементів між ними
    std::cout << "Відстань: " << diff << std::endl;  // 3

    return 0;
}

23.2 Підсумки лекції

  • ptr++ — перехід до наступного елемента
  • ptr + n — зсув на n елементів
  • ptr2 - ptr1 — кількість елементів між ними
