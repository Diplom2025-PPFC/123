Лекція 32: Геттери та сеттери

Методи доступу, валідація даних, const методи

32.1 Концепція геттерів та сеттерів

Геттери (get) повертають значення приватних полів, сеттери (set) встановлюють значення з валідацією.

#include <iostream>
#include <string>

class Person {
private:
    std::string name;
    int age;
    double salary;

public:
    // Геттери (const — не змінюють об'єкт)
    std::string getName() const { return name; }
    int getAge() const { return age; }
    double getSalary() const { return salary; }

    // Сеттери з валідацією
    void setName(const std::string& newName) {
        if (!newName.empty()) {
            name = newName;
        }
    }

    void setAge(int newAge) {
        if (newAge >= 0 && newAge <= 150) {
            age = newAge;
        } else {
            std::cout << "Некоректний вік!" << std::endl;
        }
    }

    void setSalary(double newSalary) {
        if (newSalary >= 0) {
            salary = newSalary;
        }
    }
};

int main() {
    Person p;
    p.setName("Ivan");
    p.setAge(25);
    p.setSalary(50000);

    std::cout << p.getName() << ", " << p.getAge() << " років" << std::endl;

    p.setAge(200);  // Валідація відхилить це значення

    return 0;
}

32.2 Підсумки лекції

  • Геттери — читання приватних полів
  • Сеттери — запис з валідацією
  • const методи гарантують незмінність
