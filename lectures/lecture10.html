Лекція 10: Тернарний оператор

Коротка форма if-else, вкладені тернарні оператори, області застосування

⏱️ Тривалість: 3 години 📖 Лекція 10 з 60

Рис. 10.1 — Тернарний оператор ?:

10.1 Синтаксис тернарного оператора

Тернарний оператор ?: — це скорочена форма if-else, що повертає значення. Це єдиний оператор у C++, який приймає три операнди.

// Синтаксис:
// умова ? вираз_if_true : вираз_if_false

// Звичайний if-else:
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

// Тернарний оператор:
int max = (a > b) ? a : b;

10.2 Базові приклади

#include <iostream>

int main() {
    int a = 10, b = 20;

    // Знаходження максимуму
    int max = (a > b) ? a : b;
    std::cout << "Максимум: " << max << std::endl;

    // Знаходження мінімуму
    int min = (a < b) ? a : b;
    std::cout << "Мінімум: " << min << std::endl;

    // Перевірка на парність
    int number = 15;
    std::string result = (number % 2 == 0) ? "парне" : "непарне";
    std::cout << number << " — " << result << std::endl;

    // Абсолютне значення
    int x = -10;
    int abs = (x < 0) ? -x : x;
    std::cout << "|" << x << "| = " << abs << std::endl;

    return 0;
}

10.3 Вкладені тернарні оператори

#include <iostream>

int main() {
    int a = 10, b = 20, c = 15;

    // Знаходження максимуму з трьох чисел
    int max = (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);
    std::cout << "Максимум: " << max << std::endl;

    // Оцінка за балами
    int score = 75;
    char grade = (score >= 90) ? 'A' :
                 (score >= 80) ? 'B' :
                 (score >= 70) ? 'C' :
                 (score >= 60) ? 'D' : 'F';
    std::cout << "Оцінка: " << grade << std::endl;

    return 0;
}
⚠️ Обмеження

Вкладені тернарні оператори знижують читабельність. Якщо є більше 2 рівнів вкладеності — краще використовувати if-else if або switch.

10.4 Тернарний оператор у виразах

#include <iostream>

int main() {
    int a = 10, b = 20;

    // У виведенні
    std::cout << "Більше число: " << ((a > b) ? a : b) << std::endl;

    // У присвоєнні
    int x = 5;
    x += (x > 0) ? 10 : -10;  // x = 15

    // Як аргумент функції
    auto printSign = [](int n) {
        std::cout << n << " — " << ((n > 0) ? "додатне" : (n < 0) ? "від'ємне" : "нуль") << std::endl;
    };

    printSign(10);
    printSign(-5);
    printSign(0);

    return 0;
}

10.5 Підсумки лекції

  • Тернарний оператор ?: — скорочена форма if-else
  • Повертає значення, тому можна використовувати у виразах
  • Допустимі вкладені тернарні оператори (з обережністю)
  • Не зловживайте — читабельність важливіша за короткість
