Лекція 24: Вказівники та масиви

Взаємозв'язок вказівників і масивів, передача масивів

⏱️ Тривалість: 3 години 📖 Лекція 24 з 60

24.1 Вказівники як масиви

У C++ ім'я масиву є вказівником на його перший елемент. Масиви та вказівники тісно пов'язані.

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int* ptr = arr;

    // Еквівалентні способи доступу:
    std::cout << arr[2] << std::endl;    // 30
    std::cout << *(arr + 2) << std::endl; // 30
    std::cout << ptr[2] << std::endl;     // 30
    std::cout << *(ptr + 2) << std::endl; // 30

    // Перебір через вказівник
    for (int* p = arr; p < arr + 5; p++) {
        std::cout << *p << " ";
    }
    std::cout << std::endl;

    return 0;
}

24.2 Підсумки лекції

  • Ім'я масиву — вказівник на перший елемент
  • arr[i] еквівалентно *(arr + i)
  • Вказівники дозволяють ефективно перебирати масиви
