Лекція 36: Типи наслідування

public, protected, private наслідування, множинне наслідування

⏱️ Тривалість: 3 години 📖 Лекція 36 з 60

36.1 Три типи наслідування

Типpublic в базовомуprotected в базовому
publicpublicprotected
protectedprotectedprotected
privateprivateprivate
class Base {
public: int x;
protected: int y;
private: int z;
};

class PublicDerived : public Base {
    // x — public, y — protected, z — недоступно
};

class ProtectedDerived : protected Base {
    // x — protected, y — protected, z — недоступно
};

class PrivateDerived : private Base {
    // x — private, y — private, z — недоступно
};

36.2 Підсумки лекції

  • public inheritance — найпоширеніший
  • protected — рідко використовується
  • private — реалізація через наслідування
