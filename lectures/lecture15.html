Лекція 15: Функції: оголошення та виклик

Прототипи, визначення, параметри, повернення значень

⏱️ Тривалість: 3 години 📖 Лекція 15 з 60

15.1 Що таке функція?

Функція — це іменований блок коду, який можна викликати багаторазово. Функції дозволяють уникнути дублювання коду, структурувати програму та робити її зрозумілішою.

// Структура функції:
// тип_повернення ім'я(параметри) {
//     тіло функції
//     return значення; (якщо функція не void)
// }

// Приклад простої функції:
#include <iostream>

// Визначення функції
void greet() {
    std::cout << "Привіт, світ!" << std::endl;
}

// Функція з параметром
void greetPerson(std::string name) {
    std::cout << "Привіт, " << name << "!" << std::endl;
}

// Функція, що повертає значення
int add(int a, int b) {
    return a + b;
}

int main() {
    greet();                    // Виклик функції
    greetPerson("Ivan");        // Виклик з аргументом

    int result = add(5, 3);     // Отримання результату
    std::cout << "5 + 3 = " << result << std::endl;

    return 0;
}

15.2 Прототипи функцій

Прототип (оголошення) функції повідомляє компілятору про існування функції до її визначення. Це дозволяє викликати функцію до її визначення.

#include <iostream>

// Прототипи (оголошення)
int add(int a, int b);
int subtract(int a, int b);
void printResult(int value);

int main() {
    int x = 10, y = 5;
    int sum = add(x, y);           // Виклик
    int diff = subtract(x, y);     // Виклик
    printResult(sum);              // Виклик
    printResult(diff);             // Виклик
    return 0;
}

// Визначення функцій
int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

void printResult(int value) {
    std::cout << "Результат: " << value << std::endl;
}

15.3 Передача параметрів

// Передача за значенням (копія)
void incrementByValue(int x) {
    x++; // Змінюємо лише копію!
}

// Передача за посиланням
void incrementByReference(int& x) {
    x++; // Змінюємо оригінал!
}

// Передача за вказівником
void incrementByPointer(int* x) {
    if (x != nullptr) {
        (*x)++; // Розіменування та інкремент
    }
}

#include <iostream>
int main() {
    int a = 5;

    incrementByValue(a);
    std::cout << "Після byValue: " << a << std::endl;    // 5 (не змінилось!)

    incrementByReference(a);
    std::cout << "Після byReference: " << a << std::endl; // 6

    incrementByPointer(&a);
    std::cout << "Після byPointer: " << a << std::endl;   // 7

    return 0;
}

15.5 Підсумки лекції

  • Функція — іменований блок коду для повторного використання
  • Прототипи дозволяють викликати функції до їх визначення
  • Передача за значенням — копія, за посиланням — оригінал
  • void — функція не повертає значення
