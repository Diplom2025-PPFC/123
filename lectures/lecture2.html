2.1 Компілятори C++

Для початку роботи з C++ необхідно встановити компілятор. Існує кілька популярних варіантів:

GCC (GNU Compiler Collection)

Найпопулярніший компілятор з відкритим вихідним кодом. Підтримує всі сучасні стандарти C++ (C++11/14/17/20/23). Доступний для Linux, macOS та Windows (через MinGW або MSYS2).

// Перевірка версії GCC
g++ --version

// Компіляція з вказівкою стандарту
g++ -std=c++17 program.cpp -o program

Clang

Компілятор на базі LLVM, відомий своїми чіткими та зрозумілими повідомленнями про помилки. Основний компілятор для macOS (частина Xcode). Дуже швидка компіляція та висока якість оптимізації.

Microsoft Visual C++ (MSVC)

Вбудований компілятор у Visual Studio. Найкращий вибір для розробки під Windows. Підтримує всі сучасні стандарти C++. Інтегрується з Windows SDK для створення Windows Forms додатків.

MinGW-w64

Мінімалістичний GNU для Windows — порт GCC для Windows. Легкий у встановленні, не потребує важкої інфраструктури як MSYS2. Рекомендується для початківців.

2.2 Встановлення на Windows

Спосіб 1: Visual Studio (рекомендується)

  1. Завантажте Visual Studio Community з visualstudio.microsoft.com
  2. Під час встановлення оберіть "Розробка класичних додатків на C++"
  3. Додатково оберіть "Windows Forms" для розділу Windows Forms
  4. Дочекайтеся завершення встановлення (зазвичай 10-30 ГБ)
// Перевірка встановлення у командному рядку Developer Command Prompt
cl
// Має вивести версію Microsoft C++ компілятора

Спосіб 2: MSYS2 + MinGW-w64

  1. Завантажте MSYS2 з msys2.org
  2. Встановіть та відкрийте MSYS2 terminal
  3. Виконайте: pacman -S mingw-w64-x86_64-gcc
  4. Додайте C:\msys64\mingw64in у PATH
# У MSYS2 терміналі
pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-gdb

# Перевірка
g++ --version
gdb --version

Спосіб 3: Code::Blocks (простий варіант)

Code::Blocks — легке IDE з вбудованим MinGW. Ідеально для навчання:

  1. Завантажте версію "codeblocks-XX.XXmingw-setup.exe"
  2. Встановіть — компілятор вже включено
  3. Створіть новий проект: File → New → Project → Console Application

2.3 Встановлення на Linux (Ubuntu/Debian)

# Оновлення пакетів
sudo apt update

# Встановлення компілятора та інструментів
sudo apt install build-essential gdb

# Додаткові бібліотеки для розробки
sudo apt install cmake git valgrind

# Перевірка
g++ --version
gdb --version

2.4 Встановлення на macOS

# Встановлення Homebrew (якщо не встановлено)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Встановлення Xcode Command Line Tools

# Або через Homebrew
brew install gcc gdb

2.5 IDE (Інтегровані середовища розробки)

Visual Studio (Windows)

Найпотужніше IDE для C++. Містить IntelliSense (автодоповнення), вбудований дебагер, профайлер, дизайнер форм для Windows Forms.

Visual Studio Code (кросплатформне)

Легкий редактор з розширеннями. Для C++ потрібно встановити:

  • Розширення "C/C++" від Microsoft
  • Розширення "C/C++ Extension Pack"
  • Налаштувати tasks.json та launch.json
// .vscode/tasks.json — завдання збірки
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Build C++",
      "type": "shell",
      "command": "g++",
      "args": ["-g", "-std=c++17", "${file}", "-o", "${fileDirname}/${fileBasenameNoExtension}"],
      "group": { "kind": "build", "isDefault": true }
    }
  ]
}

CLion (JetBrains)

Професійне IDE на базі CMake. Розумне автодоповнення, рефакторинг, вбудований термінал. Платне, але доступне безкоштовно для студентів.

Qt Creator

Спеціалізоване IDE для розробки з Qt framework. Чудовий дизайнер GUI, підтримка CMake та qmake.

Після встановлення компілятора створіть файл hello.cpp:

#include <iostream>

int main() {
    std::cout << "Hello, C++ World!" << std::endl;
    return 0;
}

2.7 Важливі прапорці компілятора

# Вказівка стандарту C++
g++ -std=c++17 program.cpp -o program
g++ -std=c++20 program.cpp -o program

# Включення всіх попереджень (рекомендується)
g++ -Wall -Wextra program.cpp -o program

# Відладкова інформація
g++ -g program.cpp -o program

# Оптимізація
# -O0 — без оптимізації (за замовчуванням)
# -O1 — базова оптимізація
# -O2 — хороша оптимізація (рекомендується для release)
# -O3 — максимальна оптимізація
g++ -O2 program.cpp -o program

# Комбінація (типова для розробки)
g++ -std=c++17 -Wall -Wextra -g program.cpp -o program

2.8 Відлагодження програм

Відлагоджувач (debugger) — незамінний інструмент для пошуку помилок:

Використання GDB

# Компіляція з відладковою інформацією
g++ -g program.cpp -o program

# Запуск у відлагоджувачі
gdb ./program

# Основні команди GDB:
# (gdb) break main    — встановити точку зупинки у main
# (gdb) run           — запустити програму
# (gdb) next          — виконати наступний рядок
# (gdb) step          — зайти у функцію
# (gdb) print x       — вивести значення змінної x
# (gdb) continue      — продовжити виконання
# (gdb) quit          — вийти з GDB

Відлагодження у Visual Studio

  1. Встановіть точку зупинки (F9 або клік ліворуч від рядка)
  2. Запустіть відлагодження (F5)
  3. Використовуйте F10 (Step Over), F11 (Step Into)
  4. Дивіться значення змінних у вікні Autos/Locals
  5. Використовуйте Immediate Window для обчислення виразів
💡 Порада професіонала

Звикайте компілювати з прапорцями -Wall -Wextra -Wpedantic з самого початку. Ці попередження допоможуть вам писати чистіший код та уникнути багатьох типових помилок.
