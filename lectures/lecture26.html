Лекція 26: Посилання (references)

Оголошення посилань, відмінності від вказівників, const references

⏱️ Тривалість: 3 години 📖 Лекція 26 з 60

26.1 Що таке посилання?

Посилання — це псевдонім (альтернативне ім'я) для існуючої змінної. Посилання завжди ініціалізується при створенні і не може бути змінено.

#include <iostream>

int main() {
    int x = 10;
    int& ref = x;  // ref — посилання на x

    std::cout << "x = " << x << std::endl;
    std::cout << "ref = " << ref << std::endl;

    ref = 20;  // Зміна через посилання
    std::cout << "x = " << x << std::endl;  // 20

    return 0;
}

26.2 Посилання vs Вказівники

КритерійПосиланняВказівник
ІніціалізаціяОбов'язковаМоже бути nullptr
ПерепризначенняНіТак
СинтаксисПрямий (як змінна)Потребує *
Null значенняНеможливеnullptr

26.3 Підсумки лекції

  • Посилання — псевдонім для змінної
  • Повинні бути ініціалізовані при створенні
  • Не можуть бути null
  • Використовуються для передачі параметрів за посиланням
