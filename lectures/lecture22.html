Лекція 22: Вказівники: основи

Оголошення вказівників, оператори & та *, nullptr

⏱️ Тривалість: 3 години 📖 Лекція 22 з 60

22.1 Що таке вказівник?

Вказівник — це змінна, що зберігає адресу пам'яті іншої змінної. Вказівники — одна з найпотужніших і водночас найскладніших концепцій C++.

#include <iostream>

int main() {
    int x = 42;
    int* ptr = &x;  // ptr зберігає адресу x

    std::cout << "Значення x: " << x << std::endl;
    std::cout << "Адреса x: " << &x << std::endl;
    std::cout << "Вказівник ptr: " << ptr << std::endl;
    std::cout << "Значення за адресою: " << *ptr << std::endl;

    // Зміна значення через вказівник
    *ptr = 100;
    std::cout << "Нове x: " << x << std::endl;

    return 0;
}

22.2 Оператори роботи з вказівниками

ОператорНазваОпис
&АдресаПовертає адресу змінної
*РозіменуванняДоступ до значення за адресою
->СтрілкаДоступ до членів через вказівник

22.3 Підсумки лекції

  • Вказівник зберігає адресу пам'яті
  • & — отримання адреси, * — розіменування
  • nullptr — безпечний нульовий вказівник (C++11)
