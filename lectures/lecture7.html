Порівняння значень, логічні операції, таблиці істинності

Рис. 7.1 — Логічні оператори та таблиці істинності

7.1 Оператори порівняння (реляційні)

Оператори порівняння використовуються для порівняння двох значень. Результатом є булеве значення true або false.

ОператорОписПрикладРезультат
==Дорівнює5 == 5true
!=Не дорівнює5 != 3true
<Менше5 < 3false
>Більше5 > 3true
<=Менше або дорівнює5 <= 5true
>=Більше або дорівнює5 >= 6false
#include <iostream>

int main() {
    int a = 5, b = 10;

    std::cout << "a == b: " << (a == b) << std::endl;  // 0 (false)
    std::cout << "a != b: " << (a != b) << std::endl;  // 1 (true)
    std::cout << "a < b: " << (a < b) << std::endl;    // 1 (true)
    std::cout << "a > b: " << (a > b) << std::endl;    // 0 (false)
    std::cout << "a <= b: " << (a <= b) << std::endl;  // 1 (true)
    std::cout << "a >= b: " << (a >= b) << std::endl;  // 0 (false)

    return 0;
}
❌ Типова помилка!

Ніколи не плутайте == (порівняння) з = (присвоєння)!

if (x = 5)  // ПОМИЛКА: присвоєння, а не порівняння!
if (x == 5) // ПРАВИЛЬНО: порівняння

7.2 Логічні оператори

Таблиця істинності

ABA && BA || B!A

#include <iostream>

int main() {
    bool a = true, b = false;

    std::cout << "a && b: " << (a && b) << std::endl;  // false
    std::cout << "a || b: " << (a || b) << std::endl;  // true
    std::cout << "!a: " << (!a) << std::endl;          // false
    std::cout << "!b: " << (!b) << std::endl;          // true

    // Практичний приклад: перевірка діапазону
    int age = 25;
    bool isAdult = (age >= 18 && age <= 65);
    std::cout << "Працездатний вік: " << isAdult << std::endl;

    // Складні умови
    int score = 85;
    bool isPassing = (score >= 60 && score < 90);
    bool isExcellent = (score >= 90 || score == 100);

    return 0;
}

7.3 Коротке замикання (Short-circuit evaluation)

C++ використовує коротке замикання для логічних операторів:

  • &&: якщо лівий операнд false, правий не обчислюється
  • ||: якщо лівий операнд true, правий не обчислюється
int x = 0;

// Коротке замикання — x++ НЕ виконається, бо перший операнд false
if (x != 0 && 10 / x > 5) {
    // Цей код безпечний від ділення на нуль!
}

// Без короткого замикання це призвело б до аварії!
// if (x != 0 & 10 / x > 5)  // ПОМИЛКА: ділення на нуль!

7.4 Побітові оператори

ОператорОпераціяПриклад
&Побітове І5 & 3 = 1 (0101 & 0011 = 0001)
|Побітове АБО5 | 3 = 7 (0101 | 0011 = 0111)
^Побітове XOR5 ^ 3 = 6 (0101 ^ 0011 = 0110)
~Побітове НЕ~5 = -6
<<Зсув вліво5 << 1 = 10 (0101 -> 1010)
>>Зсув вправо5 >> 1 = 2 (0101 -> 0010)
#include <iostream>
#include <iomanip>

int main() {
    unsigned char flags = 0b00001101; // 13

    // Перевірка біта (маскування)
    bool bit2 = (flags & 0b00000100) != 0;  // false
    bool bit0 = (flags & 0b00000001) != 0;  // true

    // Встановлення біта
    flags = flags | 0b00000010;  // Встановлюємо біт 1

    // Скидання біта
    flags = flags & ~0b00001000;  // Скидаємо біт 3

    // Інвертування біта (XOR)
    flags = flags ^ 0b00000001;  // Інвертуємо біт 0

    // Зсуви
    std::cout << "5 << 2 = " << (5 << 2) << std::endl;  // 20 (еквівалентно 5 * 4)
    std::cout << "20 >> 2 = " << (20 >> 2) << std::endl; // 5 (еквівалентно 20 / 4)

    return 0;
}

7.5 Підсумки лекції

  • Оператори порівняння: ==, !=, <, >, <=, >=
  • Логічні оператори: && (AND), || (OR), ! (NOT)
  • Коротке замикання для оптимізації та безпеки
  • Побітові оператори для роботи з бітами
  • Маскування, встановлення, скидання бітів
